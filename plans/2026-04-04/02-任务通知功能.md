# 任务时间到达通知功能实现计划

> 状态：计划阶段
> 创建时间：2026-04-04

## 背景

用户需要在任务到达预定时间(`start_time`)时显示浏览器系统通知。目前项目只有音频通知(`audio.js`)，没有视觉/系统通知机制。项目已有的 `pomodoroStore` 使用 `setInterval` 每秒更新 `now.value`，可以被复用。

---

## 实现方案

### 1. 创建 useNotifications.js Composable

**新建**：`src/composables/useNotifications.js`

遵循项目 Composables 模式（参考 `useDayData.js`）:
- 使用模块级 `ref` 存储共享状态
- 封装 Web Notification API
- 复用 `pomodoroStore.now` 计时器

**核心逻辑**：
```javascript
import { ref, watch } from 'vue'
import { usePomodoroStore } from '@/stores/pomodoroStore'
import { playSuccessSound } from '@/utils/audio'

// 模块级状态 - 跨组件共享单例
const notifiedTaskIds = ref(new Set())
const notificationPermission = ref('default')

export function useNotifications() {
    const pomodoroStore = usePomodoroStore()

    // 请求通知权限
    const requestPermission = async () => {
        if (!('Notification' in window)) return false
        if (Notification.permission === 'granted') {
            notificationPermission.value = 'granted'
            return true
        }
        if (Notification.permission !== 'denied') {
            const permission = await Notification.requestPermission()
            notificationPermission.value = permission
            return permission === 'granted'
        }
        return false
    }

    // 显示系统通知
    const showNotification = (title, options = {}) => {
        if (notificationPermission.value !== 'granted') return
        const notification = new Notification(title, {
            icon: '/favicon.ico',
            tag: options.tag || 'task-notification',
            ...options
        })
        notification.onclick = () => {
            window.focus()
            notification.close()
        }
    }

    // 检查任务时间并触发通知
    const checkAndNotify = (items) => {
        const now = new Date(pomodoroStore.now)
        const currentHour = now.getHours()
        const currentMinute = now.getMinutes()
        const currentDateStr = now.toISOString().split('T')[0]

        items.forEach(item => {
            if (!item.startHour || notifiedTaskIds.value.has(item.id)) return

            const [hours, minutes] = item.time.split(':').map(Number)
            const itemDateStr = item.original?.day
                ? new Date(item.original.day).toISOString().split('T')[0]
                : new Date(item.original.start_time).toISOString().split('T')[0]

            // 时间匹配检查（精确到分钟）
            if (itemDateStr === currentDateStr &&
                hours === currentHour &&
                minutes === currentMinute) {

                showNotification(`任务开始: ${item.title}`, { tag: `task-${item.id}` })
                notifiedTaskIds.value.add(item.id)
                playSuccessSound()
            }
        })
    }

    // 启动监听（由 DayView 调用）
    const startListening = (getScheduleItems) => {
        watch(
            () => pomodoroStore.now,
            () => {
                if (notificationPermission.value !== 'granted') return
                checkAndNotify(getScheduleItems())
            }
        )
    }

    // 日期切换时清空已通知记录
    const clearNotifiedHistory = () => {
        notifiedTaskIds.value.clear()
    }
}
```

### 2. 修改 App.vue - 初始化通知权限

**修改**：`src/App.vue`

在 `onMounted` 中登录后静默请求通知权限：

```javascript
import { useNotifications } from '@/composables/useNotifications'

onMounted(async () => {
    // ... 现有逻辑 ...
    if (authStore.userId) {
        const { requestPermission } = useNotifications()
        await requestPermission()
    }
})
```

### 3. 修改 DayView - 启动通知监听

**修改**：`src/views/day/index.vue`

```javascript
import { useNotifications } from '@/composables/useNotifications'

const { startListening, clearNotifiedHistory } = useNotifications()

onMounted(() => {
    // 启动通知监听
    startListening(() => dailySchedule.value)
})

// 日期切换时清空已通知记录
watch(() => dateStore.currentDate, () => {
    clearNotifiedHistory()
})
```

## 验证方式

1. `pnpm dev` - 开发服务器正常启动
2. 浏览器会弹出通知权限请求，确认授权
3. 创建一个 `start_time` 为未来几分钟的任务
4. 时间到达时，确认：
   - 系统通知显示在操作系统通知中心
   - 同时播放提示音
5. 切换日期后回到同一天，确认同一任务不会重复通知
