# 日报显示昨日改进 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 今天首次进入日视图弹出每日日报时，展示昨天日总结中填写的“改进之处”。

**Architecture:** 复用现有 `summary` 服务读取昨天的 `daily` 总结，在 Day 模块的 `useDailyReport` 中把 `content.improve` 聚合进日报状态。`DailyReportModal` 只负责按需展示文本，不直接访问数据库或 Summary 模块状态。

**Tech Stack:** Vue 3 `<script setup>`、TypeScript、Vite、Vitest、Vue Test Utils、Supabase 服务封装。

- [ ] **Step 3: Add helper to fetch yesterday improve text**

Add this function above `buildStats`:

```ts
    /**
     * 读取昨日日总结中的改进之处
     * 失败时返回空字符串，避免影响日报主体统计展示
     */
    const fetchYesterdayImprove = async (yesterday: Date): Promise<string> => {
        try {
            const yesterdayKey = toDateOnly(yesterday)
            const yesterdaySummary = await db.summary.getByDateKind(yesterdayKey, 'daily')
            const improve = yesterdaySummary?.content?.improve
            return typeof improve === 'string' ? improve.trim() : ''
        } catch (e) {
            console.warn('获取昨日改进提醒失败:', e)
            return ''
        }
    }
```

Create `src/views/day/components/__tests__/DailyReportModal.spec.ts` with:

```ts
// @vitest-environment jsdom
import { describe, expect, it } from 'vitest'
import { mount } from '@vue/test-utils'
import DailyReportModal from '../DailyReportModal.vue'

// stub Dialog 和 Button 组件，只保留插槽内容，专注验证日报内容渲染
const stubs = {
  Dialog: { template: '<div><slot /></div>', props: ['open'] },
  DialogContent: { template: '<div><slot /></div>' },
  DialogTitle: { template: '<div><slot /></div>' },
  DialogDescription: { template: '<div><slot /></div>' },
  Button: { template: '<button><slot /></button>', props: ['variant', 'class'] }
}

// 构建默认日报统计，按用例覆盖部分字段
function buildStats(overrides = {}) {
  return {
    yesterdayCompleted: 1,
    yesterdayUncompleted: 2,
    todayTotal: 3,
    carryoverToToday: 1,
    yesterdayImprove: '',
    ...overrides
  }
}

describe('DailyReportModal', () => {
  it('有昨日改进内容时显示提醒区块', () => {
    const wrapper = mount(DailyReportModal, {
      props: {
        show: true,
        stats: buildStats({ yesterdayImprove: '先完成最重要的任务\n减少切换上下文' })
      },
      global: { stubs }
    })

    expect(wrapper.text()).toContain('昨日改进提醒')
    expect(wrapper.text()).toContain('先完成最重要的任务')
    expect(wrapper.text()).toContain('减少切换上下文')
  })

  it('没有昨日改进内容时不显示提醒区块', () => {
    const wrapper = mount(DailyReportModal, {
      props: {
        show: true,
        stats: buildStats({ yesterdayImprove: '' })
      },
      global: { stubs }
    })

    expect(wrapper.text()).not.toContain('昨日改进提醒')
  })
})
```
