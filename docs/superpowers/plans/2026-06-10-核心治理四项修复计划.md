# 核心治理四项修复 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修复 goalDays 日视图延后测试失败、收紧 Supabase CRUD 类型、修正 dayStore 对视图层 composable 的反向依赖，并为目标与习惯删除增加危险操作确认。

**Architecture:** 保持现有 Vue 3 + Pinia + Supabase 分层，只做边界收口。数据层通过更精确的泛型与 Payload 类型减少 `any` 扩散；Day 操作逻辑移动到 `src/composables/`，让 Store 不再依赖 `src/views/`；危险删除沿用现有 `window.confirm` 风格，后续可统一替换为 Dialog。

**Tech Stack:** Vue 3、Pinia、Supabase JS、Vitest、vue-tsc、Vite。
